# Bienvenue dans le Lab de Création de Serveur Web Local ! 🚀

Ce lab interactif vous guidera pas à pas à travers la configuration pratique de serveurs web locaux sous **Linux (Ubuntu)**.

**Ce que vous allez accomplir :**
1. Gérer les conflits de services entre Apache2 et Nginx.
2. Créer un hôte virtuel Apache2 (`VirtualHost`).
3. Modifier les ports d'écoute standard (80 vers 8080).
4. Gérer les privilèges de fichiers (`chmod`/`chown`).
5. Mapper un nom de domaine local via le fichier `/etc/hosts`.
6. Basculer la même infrastructure vers Nginx.

Cliquez sur **Start** pour commencer !
