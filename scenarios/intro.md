# Bienvenue dans le Lab de Création de Serveur Web Local ! 🚀

[cite_start]Ce lab interactif vous guidera pas à pas à travers la configuration pratique de serveurs web locaux sous **Linux (Ubuntu)**, basé scrupuleusement sur les requis de votre notice technique.

**Ce que vous allez accomplir :**
1. [cite_start]Gérer les conflits de services entre Apache2 et Nginx[cite: 28].
2. [cite_start]Créer un hôte virtuel Apache2 (`VirtualHost`)[cite: 101].
3. [cite_start]Modifier les ports d'écoute standard (80 vers 8080)[cite: 126, 127].
4. [cite_start]Gérer les privilèges de fichiers (`chmod`/`chown`)[cite: 137].
5. [cite_start]Mapper un nom de domaine local via le fichier `/etc/hosts`[cite: 142].
6. [cite_start]Basculer la même infrastructure vers Nginx[cite: 280].

Cliquez sur **Start** pour commencer !
